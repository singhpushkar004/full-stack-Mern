# OOP's
    OOP's is stands for Object Oriented programming System .
    This Concept introduced in js in ES6.
    There are mainly four pillars in OOPS .
    This system is based on class and object .
    a class is a blueprint of a object and object is a real world entity .

1. Inheritance ==> 
            Inheritance is concept in which a child class inherit the methods and properties of a parent class .
2. Encapsulation ==>
        Encapsulation is concept in which we wrap the methods and properties  in the single unit.
3. Polymorphism ==>
        Polymorphism is a concept in which a method can be used many times in a child class. Polymorphism means one thing many forms .
4. Abstraction ==>
        abstraction is a concept in which showing the essential properties and hiding the internal properties .


# Methods ==>
    Functions inside the class is called methods .
# Properties ==>
    Properties are the data member of a class .
# Constructor==>
    Constructor is a function in the class that is called itself when a object created .
